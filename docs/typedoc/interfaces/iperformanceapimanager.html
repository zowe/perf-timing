Interface IPerformanceApiManager

This interface defines the properties that a PerformanceApiManager must expose to its managed {@link PerformanceTools} instance. This interface must be implemented by any manager object passed to the {@link PerformanceTools} constructor of this package. By default, the PerformanceApiManager will implement this interface.
